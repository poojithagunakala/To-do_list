# 📝 To-Do List Web App

A simple and interactive **To-Do List application** built using **HTML, CSS, and JavaScript**. This project helps users manage daily tasks efficiently by adding, completing, and deleting tasks.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 💾 Data stored in browser (local storage)
* 🎨 Clean and responsive UI

---

## 🛠️ Technologies Used

* **HTML** – Structure of the app
* **CSS** – Styling and layout
* **JavaScript** – Functionality and interactivity

---

## 📂 Project Structure

```
📁 To-Do-List
│── index.html
│── style.css
│── script.js
```

---

## ⚙️ How It Works

1. Enter a task in the input field
2. Click the **Add** button
3. Your task will appear in the list
4. Click on a task to mark it as completed
5. Click the delete icon to remove a task
